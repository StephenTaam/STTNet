STTNet

Lightweight High-Performance C++ Network Framework

STTNet is a lightweight, high-performance server framework written in C++11. It provides full support for TCP/UDP/HTTP/WebSocket, TLS/HTTPS/WSS encryption, multithreaded epoll, file I/O, logging, JSON processing, signal management, encryption modules, information security, and more.
