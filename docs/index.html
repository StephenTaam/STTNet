STTNet

Lightweight High-Performance C++ Network Framework

STTNet is a lightweight and high-performance server framework written in modern C++11. It supports TCP/UDP/HTTP/WebSocket, TLS/HTTPS/WSS encrypted communication, multithreaded epoll, file I/O, logging, JSON processing, signal/exception handling, encryption utilities, information security modules, and more.
