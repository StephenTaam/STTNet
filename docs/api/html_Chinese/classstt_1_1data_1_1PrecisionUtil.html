static float& stt::data::PrecisionUtil::getValidFloat ( float &  number,
const int &  bit 
)
static

根据数值动态调整小数精度，保留指定数量的有效数字。

对于小于 1 的小数，会先确定其最前位有效数字所处的位置， 然后在此基础上保留 bit 位有效数字。 修改后的数值将四舍五入保留合适的小数位，并写回原变量。

参数
number待处理的 float 数值（将被修改）。
bit希望保留的有效数字位数。
返回
修改后的 float 引用。
