将普通字符串转化为对应的十六进制表示字符串（hex string）。

例如输入 "ABC" 会被转化为 "414243"，每个字符转换为其 ASCII 的十六进制形式。

参数
ori_str输入的原始字符串（可以包含任意字符，包括不可见字符）。
result存放转换后结果的字符串引用。
返回
转换后的十六进制字符串 result 的引用。
