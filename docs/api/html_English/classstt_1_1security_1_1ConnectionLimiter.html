Detailed Description

Limit the class of connections from the same IP.

Note
Without locks, synchronization and thread safety are not ensured, and they are ensured at the upper level.

requestRateThe maximum number of requests allowed per second for the same IP address (default is 12)
connectionTimeoutThe number of seconds a connection is considered a zombie connection if there is no response (in seconds). The default is 60 seconds. -1 means no limit.
